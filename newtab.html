Lumina Guide

Everything you need to know
Search

Type anything and press Enter — opens in a new tab. Click the engine label in the search bar to open a dropdown with all engines. The send button color matches your chosen engine. Change the default engine in Settings.

Focus search bar/ from anywhere on the page
AI enginesClaude, Gemini, ChatGPT, Perplexity — query is pre-filled and auto-submitted
Web enginesGoogle, Google AI Mode, Bing, DuckDuckGo
Dev enginesGitHub, Stack Overflow
Quick Links

Bookmarks shown in a 4-column grid. Click any card to open it in the current tab. All actions appear on hover.

Add linkAdd Link button (top right of grid)
ReorderDrag the ⠿ handle on the left of any card
Hover actionsOpen in new tab · Copy URL · Edit · Delete
Icon modeGrid icon button → compact favicon tiles; hover a tile for a floating action menu
Sections+ Section button → named groups; click section label to rename; assign links via Edit
Bookmark syncBookmarks button → choose a Chrome folder; enable auto-sync or merge mode. Sync is bidirectional: adding, editing, or deleting a link also updates Chrome bookmarks
ExportExport button → Markdown, Apple Notes, or plain text
Notes

Open the notepad icon (top bar) and select the Notes tab. Rich-text editor — everything saves automatically on every keystroke.

Multiple notes+ button in the tab row adds a new note; × on a tab deletes it; click tabs to switch

Saved Links

A personal reading list. Open the notepad icon → Saved Links tab. The panel can be Light, Dark, or System themed independently of the background.

Add from panel+ Add Link button inside the tab
Save current tabClick the Lumina icon in Chrome's toolbar from any page
TagsAssign color-coded tags; 8 preset colors; create custom tags
Filter by tagClick a tag chip in the filter row
SortNewest first or A – Z
Hover actionsCopy URL · Delete
Weather

Displayed in the top-left corner. Settings → Weather to configure.

Auto location"Use my location" toggle — browser asks permission once
Postal overrideEnter a zip/postal code to lock to a specific location
Units°F / °C toggle in Settings
Refresh rateEvery 30 minutes; cached between refreshes
Data sourceOpen-Meteo — free, no API key needed
Themes & Wallpapers

Settings → Background. Switch between Colors and Wallpaper tabs. Selected themes or wallpapers rotate with each new tab open.

Color themesCosmic · Aurora · Sunset · Ocean · Forest · Ember · Minimal · Candy — select multiple to rotate
IntensitySubtle / Medium / Vivid
Dynamic wallpapersBing Daily · NASA APOD · Wikimedia POTD — refreshed each day, rotate each tab
Custom wallpaperUpload one or more images; filename becomes the title; edit emoji/title on the card. Only titles and selection sync (not image data), so the gist stays small.
Panel themeNotes/Saved Links panel: Dark · Light · System (follows OS)
Daily FocusSettings → Daily Focus — quotes rotate daily; fetch extras from ZenQuotes
GreetingSettings → Options — add your name for "Good morning, Name" or switch to a fully custom greeting
Auto-fill forms

Add entries in Settings → Auto-fill forms. Then right-click on any page and choose Auto-fill form with → an entry to fill inputs (name, email, phone, address, etc.). Entries sync with your gist.

Sync via GitHub Gist

All data syncs across browsers (Chrome, Firefox, Safari, Edge) using a private GitHub Gist. Settings → Sync.

SetupCreate a GitHub PAT with gist scope → paste into Settings → Sync
First useCreates one main gist (settings, links) and a separate gist per note — each note appears as its own gist in your GitHub Gists list
Auto-push3 seconds after any change (links, settings, and each note as its own file)
Auto-pullOn every new tab open
ManualSync Now button in Settings
No tokenExtension works normally without sync — nothing changes
