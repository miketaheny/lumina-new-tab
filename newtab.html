Lumina Guide

Everything you need to know
Search

Type anything and press Enter — opens in a new tab. Click the engine label in the search bar to open a dropdown with all engines. The send button color matches your chosen engine. Change the default engine in Settings.

Focus search bar/ from anywhere on the page
AI enginesClaude, Gemini, ChatGPT, Perplexity — query is pre-filled and auto-submitted
Web enginesGoogle, Google AI Mode, Bing, DuckDuckGo
Dev enginesGitHub, Stack Overflow

Auto-fill forms

Add entries in Settings → Auto-fill forms. Then right-click on any page and choose Auto-fill form with → an entry to fill inputs (name, email, phone, address, etc.). Entries sync with your vault.

Sync via Obsidian (Local REST API)

Notes sync as .md files in your Obsidian vault's Lumina/ folder via the Local REST API plugin. Settings → Sync.

SetupInstall the Local REST API plugin in Obsidian → copy the API key → paste URL and key into Settings → Sync
First useCreates a Lumina/ folder in your vault with each note as a .md file and a .lumina-sync.json settings file
Auto-push3 seconds after any change (links, settings, and each note as its own .md file). Also flushes immediately when you switch tabs, switch apps, or close the window.
Auto-pullWhen you return to the Lumina tab or refocus the Chrome window, vault edits are pulled in automatically so notes you changed in Obsidian don't get overwritten.
ConflictsIf a note changed in both places since the last sync, a banner lets you choose which version to keep.
ManualSync Now button in Settings does a full pull, including imports for new files and deletions for files removed from the vault.
Not configuredExtension works normally without sync — nothing changes
